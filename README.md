# PM Claw

Персональный ассистент продакт-менеджера на базе OpenClaw-агента.

## Структура проекта

```
src/           — основной код агента
src/skills/    — навыки агента (skills)
src/config/    — конфигурация
tests/         — тесты
docs/          — документация
```

## Запуск

```bash
pip install -r requirements.txt
python -m src.main
```
