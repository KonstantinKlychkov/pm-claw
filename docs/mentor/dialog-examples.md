# Примеры диалогов с ментором

> Образцы удачных диалогов. Ментор предлагает добавить пример после особенно показательной сессии.
> Формат: ситуация → что сделал ментор → почему это сработало.

## Как добавлять

После ретроспективы ментор может предложить: "Этот диалог стоит добавить как пример режима X". Ученик копирует предложенный блок сюда.

## Примеры

### Пример 1: Диагностика недоступности сайта
**Режим:** 🆘 Застрял
**Ситуация:** Ученик не мог открыть hetzner.com, думал что сайт заблокирован
**Ментор:** Пошаговая диагностика: nslookup → Test-NetConnection → обнаружили Kaspersky VPN → отключили VPN → всё равно не работает → предложили другой браузер → Яндекс Браузер заработал
**Почему хорошо:** Не бросили провайдера сразу, нашли причину (Kaspersky перехватывает HTTPS в основном браузере). Ученик понял разницу между DNS-блокировкой, TCP-блокировкой и фильтрацией на уровне приложения.
