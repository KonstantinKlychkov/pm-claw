# Известные пробелы ученика

> Этот файл обновляется после ретроспективы каждой фичи.
> Ментор использует его для челленджей и для калибровки объяснений.

## Активные пробелы

- **~/.claude/ vs .claude/** — путает глобальную конфигурацию (`~/.claude/`, все проекты) и проектную (`.claude/`, конкретный проект). Напоминай при работе с конфигами.
- **JSON в PowerShell CLI** — на Windows PowerShell плохо экранирует JSON в командной строке. Предлагай редактирование файлов напрямую.
- **Glob-scoped rules** — загружаются условно, не всегда работают как ожидается. Было слабым местом в тестах курса.
- **Linux-администрирование** — базовые навыки подзабыты: управление пользователями, пароли, sudo. На практике встретились sudo visudo, systemd (systemctl enable/restart/status), journalctl — команды подсказывал ментор. Напоминай команды при работе с сервером.
- **python -m vs прямой запуск** — столкнулся с ошибкой резолвинга пакетов. Запомни: для запуска модулей внутри пакета используй python -m src.bot_runner, не python src/bot_runner.py

## Закрытые пробелы

- **SSH-ключи и passphrase** — разобрался на практике: passphrase ключа ≠ пароль пользователя. Создал новый ключ без passphrase, добавил на сервер вручную.
