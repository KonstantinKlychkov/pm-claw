# Известные пробелы ученика

> Этот файл обновляется после ретроспективы каждой фичи.
> Ментор использует его для челленджей и для калибровки объяснений.

## Активные пробелы

- **~/.claude/ vs .claude/** — путает глобальную конфигурацию (`~/.claude/`, все проекты) и проектную (`.claude/`, конкретный проект). Напоминай при работе с конфигами.
- **JSON в PowerShell CLI** — на Windows PowerShell плохо экранирует JSON в командной строке. Предлагай редактирование файлов напрямую.
- **Glob-scoped rules** — загружаются условно, не всегда работают как ожидается. Было слабым местом в тестах курса.
- **Linux-администрирование** — базовые навыки подзабыты: управление пользователями, пароли, sudo. Напоминай команды при работе с сервером.
- **SSH-ключи и passphrase** — путал passphrase ключа с паролем пользователя. При генерации ключей уточняй, нужен ли passphrase.
