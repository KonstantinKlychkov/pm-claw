# Пайплайн разработки PM Claw

> Ментор ведёт ученика по этим шагам в режиме "Новая фича".
> Называй текущий шаг. Если ученик пропускает шаг — напомни.
> После каждой фичи — оцени, какой шаг автоматизировать следующим.

---

## Уровни автономности

| Уровень | Значение | Кто делает |
|---------|----------|------------|
| 🔴 Ручной | Ученик делает всё сам | Человек |
| 🟡 Полуавто | Инструмент помогает, но ученик запускает и контролирует | Человек + Claude Code |
| 🟢 Авто | Claude Code делает сам, ученик только проверяет результат | Claude Code |

---

## Шаги

### 1. Идея
**Уровень:** 🔴 Ручной
**Что происходит:** обсуждение scope, создание GitHub Issue
**Инструменты:** чат с ментором, GitHub MCP
**Автоматизировано:** —

### 2. Проектирование
**Уровень:** 🔴 Ручной
**Что происходит:** архитектура, зависимости, интерфейсы
**Инструменты:** Plan Mode (Shift+Tab), Context7
**Автоматизировано:** —

### 3. Git-ветка
**Уровень:** 🟡 Полуавто
**Что происходит:** создание ветки `feature/<название>`
**Инструменты:** Claude Code CLI (git)
**Автоматизировано:** checkpoints автоматические

### 4. Реализация
**Уровень:** 🟡 Полуавто
**Что происходит:** написание кода
**Инструменты:** Claude Code CLI, Agent Teams для параллельных задач
**Автоматизировано:** —

### 5. Тесты
**Уровень:** 🟡 Полуавто
**Что происходит:** написание и запуск тестов
**Инструменты:** субагент test-writer, skill `/test-all`
**Автоматизировано:** test-writer генерирует тесты, `/test-all` запускает

### 6. Ревью
**Уровень:** 🟡 Полуавто
**Что происходит:** проверка кода
**Инструменты:** субагент code-reviewer, или ревью в чате с ментором
**Автоматизировано:** code-reviewer даёт первичный анализ

### 7. PR
**Уровень:** 🟡 Полуавто
**Что происходит:** коммит, пуш, создание PR
**Инструменты:** skill `/commit-push-pr`, GitHub MCP
**Автоматизировано:** skill собирает коммит + пуш + PR в одну команду

### 8. Мерж
**Уровень:** 🔴 Ручной
**Что происходит:** финальный review, merge, git pull
**Инструменты:** GitHub
**Автоматизировано:** —

### 9. Ретроспектива
**Уровень:** 🟡 Полуавто
**Что происходит:** анализ, обновление Project Knowledge
**Инструменты:** чат с ментором (режим "Фича завершена")
**Автоматизировано:** ментор генерирует блоки обновлений

---

## Критические проверки

- Код без ветки → напомни про шаг 3
- Коммит без тестов → напомни про шаг 5
- Мерж без ревью → напомни про шаг 6

---

## Цель

Постепенно двигать шаги от 🔴 к 🟡 и от 🟡 к 🟢.
Не автоматизировать всё сразу — одно улучшение за итерацию.
Каждое изменение уровня фиксировать в истории ниже.
